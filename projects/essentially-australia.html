Essentially Australia is a Byron Bay native essential oils brand, Est. 2012, with 38+ oils sourced from Australian botanical ingredients. Their WordPress e-commerce site had broken information architecture, invisible CTAs, a blank hero, and a purchase path that made testers say — "I'm actually gonna suffer."

I conducted a full UX audit across four competing Australian essential oil sites, identified Essentially Australia as the highest-friction villain, and redesigned the entire purchase experience from the ground up — mobile and desktop.

The Problem

Friction
every
step.

The original site failed at every stage of the purchase journey — from the hero that showed a lighthouse instead of a product, to a cart that silently accepted wrong orders without any feedback.

H2 — Real World Match
Broken IA
Oils, gift vouchers, oracle cards — all equal weight. No hierarchy, no logic.
H1 — System Status
Silent Cart
Cart accepted items with zero feedback. Users repeated actions assuming failure.
H5 — Error Prevention
Manual Cart Update
Quantity changes required a manual click. Users checked out wrong orders.
H8 — Aesthetic
Zero Hero Intent
Hero showed a lighthouse. No product, no CTA, no value. Immediate mismatch.
Process

How I
got there.

  • 01
    UX Audit — Visual Forensics
    Analysed 4 Australian essential oil sites across 7 elements — typography, grid, visual hierarchy, colour, WCAG, dark patterns, responsive breaks. Verified contrast ratios in DevTools.
  • 02
    Nielsen's Heuristics + Cognitive Walkthrough
    Applied 6 heuristics across all 4 sites. Ran a think-aloud cognitive walkthrough — "Find a food grade pepper oil, check sizes, add to cart." 5 key friction moments recorded.
  • 03
    Moodboard + Art Direction
    Developed 3 art directions — Botanical (chosen), Brutalist (rejected), Swiss Grid (rejected). Palette extracted from Byron Bay's volcanic hinterland. Colour does navigation work: Uplift, Calm, Restore, Ground.
  • 04
    Wireframes — Dual Path IA
    Designed two purchase paths — Browse (feeling-first) and Search (keyword). 5 layout directions sketched before committing. Full flow: Hero → Product List → Filter → Product Detail → Cart → Checkout.
  • 05
    Hi-Fi Prototype — Mobile + Desktop
    Fully interactive Figma prototypes at 390×844 (mobile) and 1440×1024 (desktop). Complete purchase flow with system feedback, feeling-category navigation, WCAG AA typography.
  • 06
    Usability Testing + Iteration
    Think-aloud with a UX Designer (3+ years). 3 tasks completed. 1 friction found — Food Grade filter buried. Iterated: promoted to standalone chip. Design validated.
The Solution

Emotion
first
design.

The redesign centres on feeling-first navigation — users shop by how they want to feel, not botanical names they don't know. Four categories: Uplift, Calm, Restore, Ground.

Colour does the navigational work before the label is read. Purchase path reduced from 8 to 4 steps. Cart gives immediate visual feedback. Size selection updates price in real time.

H6 — Recognition
Feeling-First IA
Colour dots read before labels. Expert user defaulted to feeling nav over search. No vocabulary gap.
H1 — System Status
"Successfully Added!"
CTA replaces button on add. Cart increments. Duplicates prevented.
H5 — Error Prevention
Live Price Sync
25ml selected → button reads $38.95. Selection and cost always in sync.
GenAI Feature
Scent Memory
Conversational AI maps emotional input to product. No botanical knowledge required.

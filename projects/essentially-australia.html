Overview

Essentially Australia is a Byron Bay native essential oils brand, Est. 2012, with 38+ oils. Their existing WordPress e-commerce site had broken information architecture, invisible CTAs, a blank hero, and a purchase path that made testers say — "I'm actually gonna suffer."

I conducted a full UX audit across four competing Australian essential oil sites, identified Essentially Australia as the highest-friction villain site, and redesigned the entire purchase experience from the ground up.

Process

HOW I
GOT THERE.

  • 01
    UX Audit — Visual Forensics
    Analysed 4 Australian essential oil sites across 7 elements — typography, grid, visual hierarchy, colour & brand, WCAG, dark patterns, and responsive breaks. Used DevTools to verify contrast ratios and font sizes.
  • 02
    Nielsen's Heuristics + Cognitive Walkthrough
    Applied 6 heuristics across all 4 sites. Ran a think-aloud cognitive walkthrough on Essentially Australia — "Find a food grade pepper oil, check sizes, add to cart." Recorded 5 key friction moments tied directly to heuristic violations.
  • 03
    Moodboard + Art Direction
    Developed 3 art directions — Botanical (chosen), Brutalist (rejected), Swiss Grid (rejected). Palette extracted from Byron Bay's volcanic hinterland. Colour does navigation work: Uplift, Calm, Restore, Ground.
  • 04
    Wireframes — Dual Path IA
    Designed two purchase paths — Browse (feeling-first) and Search (keyword). Sketched 5 layout directions before committing. Wireframes covered full flow: Hero → Product List → Filter → Product Detail → Cart → Checkout.
  • 05
    Hi-Fi Prototype — Mobile + Desktop
    Built fully interactive Figma prototypes at 390×844 (mobile) and 1440×1024 (desktop). Complete annotated purchase flow with system feedback, feeling-category navigation, and WCAG AA compliant typography.
  • 06
    Usability Testing + Iteration
    Think-aloud testing with a UX Designer (3+ years experience). 3 tasks completed. 1 friction point found — Food Grade filter buried. Iterated: promoted to standalone chip in "Best For" group. Design validated.
The Solution

EMOTION
FIRST
DESIGN.

The redesign centres on feeling-first navigation — users shop by how they want to feel, not by botanical names they don't know. Four categories: Uplift, Calm, Restore, Ground. Colour does the navigational work before the label is read.

Purchase path reduced from 8 steps to 4. Cart gives immediate visual feedback. Size selection updates price inside the CTA in real time. Free shipping progress bar changes behaviour — users add a second product from proximity to reward, not browse intent.

H6 — Recognition
Feeling-First IA
Colour dots read before labels. Expert user defaulted to feeling nav over search bar. No vocabulary gap.
H1 — System Status
"Successfully Added!"
CTA replaces button on add. Cart increments. Duplicates prevented. User always knows what happened.
H5 — Error Prevention
Live Price Sync
25ml selected → button reads $38.95. Selection and cost always in sync. No manual calculation.
GenAI Feature
Scent Memory
Conversational AI maps emotional input to product recommendations. No botanical knowledge required.
